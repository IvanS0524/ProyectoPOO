# Proyecto Final POO

Este repositorio contiene el **Proyecto Final** de la asignatura Programación Orientada a Objetos **(POO)**.

## Descripción

El proyecto reúne los conceptos aprendidos durante el curso de POO, tales como:
- Clases y objetos
- Herencia y polimorfismo
- Encapsulamiento y abstracción
- Interfaces
- Manejo de excepciones
- Manejo de archivos
